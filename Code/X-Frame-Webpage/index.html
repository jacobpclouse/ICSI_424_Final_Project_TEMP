We can create base cases for chromium based browsers and see how behavior differs between them. Checking if implementing X-Frame-Options messes with the user experience / slows down operations.
  • Clickjacking Defense -- Check clickjacking attacks against browsers to see how each browser defeats it and which work better than others. I was thinking that we could look at the implimentation of each and guage which ones are more soundly implimented vs which ones are more likely to leak information. We could also look back and try to see if there were any recent security issues on each browser for xframes
  • Practical Tips -- Based on what we find, documenting practical tips for developers on how to use X-Frame-Options, what works and what doesn't. This will be written up in the form of a after action report.
